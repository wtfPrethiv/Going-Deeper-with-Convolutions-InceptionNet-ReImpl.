# Attention-is-all-you-need-ReImpl.

A from-scratch PyTorch implementation of the original Transformer architecture described in the landmark 2017 paper, *"Attention Is All You Need"* (Vaswani et al.). 

This project is built from the ground up to provide a deep, foundational understanding of how self-attention mechanisms, positional encodings, and encoder-decoder stacks function under the hoooood 🌿🚬.

## Stuffffffffff (Work in Progress)

* **Input Embeddings:** Custom scaled token embedding layer.
* **Positional Encoding:** Implementation of sine and cosine absolute positional encodings.
* **Multi-Head Attention:** Scaled dot-product attention with masking capabilities.
* **Transformer Encoder:** Complete with Position-wise Feed-Forward Networks, Layer Normalization, and residual (Add & Norm) connections.
